# 조각 도서관 Sculpture Library

조각에 관한 글모음. 조각사 수업 내용을 참고한 도서 목록 웹페이지.

## 텍스트 자동 대비 (text-contrast)

배경의 갈색 삼각형 위에 있는 텍스트는 흰색, 흰 배경 위 텍스트는 검정색으로 자동 전환된다. `mix-blend-mode: difference`와 유사하지만, 결과가 항상 순수한 흑/백으로만 나온다.

### 원리

```
  1. 배경                2. 그라디언트             3. 글자로 자르기
                            (background)          (background-clip: text)

  +----------+/          +----------+/            +----------+/
  |          /           |##########/             |  Hello   /
  |  흰색    /갈색         |##검정###/흰색|           |##Hel##/lo
  |        / 배경         |########/   |           |##Sc##/ul |
  |       /    |         |#######/    |           |####/ptu  |
  |      /     |         |######/     |           |###/ re   |
  +-----/------+         +-----/------+           +---/------+

                         # = 검정 영역               ## = 검정 글자
                           = 흰색 영역                 = 흰색 글자
```

화면 전체에 대각선을 따라 검정/흰색이 딱 나뉘는 그라디언트를 깔아둔다.
그 위에 `background-clip: text`를 적용하면 글자 모양으로만 그라디언트가 보인다.
갈색 배경 위의 글자는 흰 부분이, 흰 배경 위의 글자는 검정 부분이 드러난다.

### 추가한 파일

- `text-contrast.css` — 텍스트에 뷰포트 고정 그라디언트 적용
- `text-contrast.js` — 대각선 각도/위치 계산 및 테두리 색상 처리
